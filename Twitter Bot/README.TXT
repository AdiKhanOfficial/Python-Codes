Tweet Copy from User's Timeline 
This code is written to Copy the Tweets from a Specific User's Timeline and Post That On Your Timeline.

BASIC SETTINGS:
Add Customer Key in api_key
Add Customer Key Secret in api_key_secret
Add Access Token in access_token
Add Access Token Secret in access_token_secret

Change Taget User name in user = 'CoinMarketCap'

Change Number of Tweets to Copy in user_tweets = get_user_tweets(user, 1)

Enjoy
