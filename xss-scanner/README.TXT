XSS-Scanner Tool is to scan a website for xss vulnerablity.
Use settings.xml file to do some basic settings like:
 1. Exclude urls, files extension or files postfixes.
 2. Set-up some cookies if you want to target a website that requires login etc. 

Just Run the xss.py file and provide the website main url. 
Procedure:
Tool will crawl all the pages and then will try to send xss payload. 
Tool will work in both method GET/POST based on your choice.
