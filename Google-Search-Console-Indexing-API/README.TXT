This tool is very useful to submit your website urls to Google Search Console for Indexing/update.
Procedure:
1. Setup your Google App. 
2. Download API Key. 
3. Put your API Key content in "apidetails.json" file.
4. Add your urls in "data.csv".
Note: This code is only for submitting urls for indexing/update not for url deletion request. Date column in "data.csv" file will be ignored.
